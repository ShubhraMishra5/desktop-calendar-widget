# Desktop Calendar Widget

A minimal desktop calendar widget built using **Electron**, **HTML**, **CSS**, and **JavaScript**.  
It displays the current **date and month** in a clean, aesthetic widget-style UI.


##  Features
- Desktop widget built with Electron
- Displays current day and month
- Custom UI design with background frames
- Fixed-size, non-resizable window
- Lightweight and fast


## Tech Stack
- Electron
- JavaScript
- HTML
- CSS
